Text-to-Speech Highlighter

functioning chrome extension

At EngHacks 2022 my team and I developed a chrome extension with 600+ users and 15 000+ monthly API calls on the chrome web store using Microsoft's Azure Translation and Language Detection API with JavaScript to provide accessibility in text to speech to and from any language on any website assisting the illiterate, dyslexic and those learning a new language.

I worked on the extension's backend using Azure Microsoft's Translation and Language Detection API.
